# Profile
minicube.netは自主制作ハードウェアの開発、作成を行っているサークルです。電子基板の設計、部品実装やレーザーカッターを使用して素材を加工しオリジナルのハードウェアを作成しています。電子部品やモジュールなど単体では使用できない部品や機材を、変換基板や変換アダプタ、ジョイントを使うことにより便利に利用するお手伝いができれば幸いです
