# Overview  
minicube.netは自主制作ハードウェアの開発、作成を行っています。  
電子基板の設計、部品実装やレーザーカッターを使用して素材や市販品を加工しオリジナルのハードウェアを作成ています。  
電子部品やモジュールなど単体では使用できない部品や機材を、変換基板や変換アダプタ、ジョイントを使うことにより便利使うお手伝いをしています。
